# Operly AI Configuration

Este diretório contém definições de agentes e skills para assistentes de IA.

## 📁 Estrutura

```
.ia/
├── agents/     # Personas/roles de agentes de IA
└── skills/     # Definições de funcionalidades/capacidades
```

## 🤖 Agents

| Agent | Responsabilidade |
|-------|------------------|
| `architecture-agent` | Design de arquitetura do sistema |
| `backend-engineer` | APIs e regras de negócio |
| `frontend-engineer` | Interface do usuário |
| `data-analyst` | Análise de dados e insights |
| `devops-engineer` | CI/CD e infraestrutura |
| `product-analyst` | Decisões de produto |
| `security-specialist` | Segurança da plataforma |

## 🛠️ Skills

| Skill | Descrição |
|-------|-----------|
| `get_business_dashboard` | Métricas do negócio |
| `get_financial_summary` | Resumo financeiro |
| `get_schedule_overview` | Visão geral da agenda |
| `manage_appointments` | CRUD de agendamentos |
| `manage_clients` | CRUD de clientes |
| `manage_services` | Catálogo de serviços |
| `manage_vehicles` | Veículos dos clientes |

## 📝 Uso

Estas definições podem ser usadas como contexto para assistentes de IA que trabalham com o Operly, definindo personas especializadas e capacidades disponíveis.
